# Calculator

A simple web-based calculator application.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean and intuitive user interface
- Quick and responsive calculations

## How to Use

1. Open `calcultor.html` in your web browser
2. Click the number and operation buttons to perform calculations
3. Press the equals button to see the result
4. Click the clear button to reset the calculator

## Files

- `calcultor.html` - Main HTML file
- `calcultor.css` - Styles and layout
- `calculator.js` - Calculator logic and functionality

## Browser Compatibility

Works on all modern web browsers that support HTML5 and JavaScript.
